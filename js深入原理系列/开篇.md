# js深入原理系列之开篇

在工作中经常需要打开浏览器调试工具调试页面，如下图所示，调试工具的中call stack、scope是什么意思？另外，前端经常会涉及闭包、原型链等词汇，那这些又是怎么回事？带着这些疑问，于是就有js深入原理系列文章。
![](images/kaipian1.png)

该系列的探索笔记主要是以google国外的blog为主，因此，文章中的参考链接有些是需要翻墙才能打开。

## 目的

通过阅读该系列文章，希望访客你能够掌握以下概念：
1. js的事件模型
2. js的oop实现原理
3. js代码如何运行的
4. js闭包的原理
5. js中this指向的原理
6. ...
